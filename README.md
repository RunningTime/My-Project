但是MFC的开发太慢而且过时，机缘巧合下寻到Qt，并有Qt编写象棋游戏实例教程，
并且同时本人见证alphago人工智能击败李世石九段，体现了人工智能的强大，有学习的必要
故此在2016年3月5日起，利用课余时间学习并编写象棋游戏，希望能在毕业之前做出功能丰富，界面漂亮的C++应用软件
